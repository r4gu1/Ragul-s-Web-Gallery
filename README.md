# Ragul's Web Gallery

A compact portfolio web gallery showcasing projects, experience, and skills for Ragul R. Built with React and styled components to be responsive and easy to extend.

**Quick Overview:**
- **Name:** Ragul R
- **Roles:** Full Stack Developer, Web Developer, Programmer
- **Repo:** `r4gu1/Ragul-s-Web-Gallery`

**Features:**
- Projects list with links to GitHub and live demos
- Education, experience timeline, and contact sections
- Responsive components and simple animations

**Tech stack:**
- React (v18), `react-scripts`
- MUI, Styled Components, React Router
- Node / NPM for local development

**Run locally (PowerShell):**
```powershell
Set-Location -Path 'D:\admin\ragul\project\Ragul-s-Web-Gallery'
npm install
npm start
```

**Build / Deploy:**
- `npm run build` then host the `build/` folder (or use `npm run deploy` if configured with `gh-pages`).
